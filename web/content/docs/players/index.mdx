# Observer for Players

Welcome! To play on an Observer-enabled server, you need to install the Observer client mod on your local game. The mod enables custom UI, ambient sounds, and environment changes.

## Prerequisites

- **Minecraft**: You must be running the specific version of Minecraft supported by the server.
- **Fabric Loader**: Observer is currently a Fabric mod.

## Installation Steps

1. Install [Fabric Loader](https://fabricmc.net/use/installer/) for the correct Minecraft version.
2. Open your Minecraft `.minecraft/mods` folder.
3. Download the `observer-fabric-<version>.jar` file provided by your server owner.
4. Place the `.jar` file inside the `mods` folder.
5. Launch the game using the Fabric profile.

## Troubleshooting

- **Server says "Observer required"**: Ensure the mod is in the `mods` folder and that you launched the Fabric profile.
- **Missing Textures (Purple/Black boxes)**: Ensure you have accepted the server's Resource Pack when joining. Observer UIs rely on server resource packs to display images.
- **Keybinds not working**: Check your Minecraft Controls menu to ensure no conflicts exist with the custom keys defined by the server.
