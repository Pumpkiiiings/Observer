# Eventos

Observer dispara eventos de Bukkit siempre que el cliente envía datos al servidor. Estos eventos te permiten detectar cuándo un jugador salta, esprinta, se agacha o interactúa con el mod de otras maneras.

Todos los eventos residen en el paquete `com.observer.paper.api.events` y heredan de `ObserverPlayerEvent`.

---

### `ObserverPlayerJumpEvent`
> Se dispara cuando el cliente inicia un salto.

`com.observer.paper.api.events.ObserverPlayerJumpEvent`

**Firma**
```java
public class ObserverPlayerJumpEvent extends ObserverPlayerEvent
```

**Métodos**

#### `getPlayer`
```java
public Player getPlayer()
```
Retorna: El jugador que saltó.

**Ejemplo**
```java
@EventHandler
public void onPlayerJump(ObserverPlayerJumpEvent event) {
    Player player = event.getPlayer();
    player.sendMessage("¡Has saltado!");
}
```

---

### `ObserverPlayerSprintEvent`
> Se dispara cuando cambia el estado de esprintar de un jugador.

`com.observer.paper.api.events.ObserverPlayerSprintEvent`

**Firma**
```java
public class ObserverPlayerSprintEvent extends ObserverPlayerEvent
```

**Métodos**

#### `getPlayer`
```java
public Player getPlayer()
```
Retorna: El jugador.

#### `isSprinting`
```java
public boolean isSprinting()
```
Retorna: `true` si comenzó a esprintar, `false` si se detuvo.

**Ejemplo**
```java
@EventHandler
public void onSprint(ObserverPlayerSprintEvent event) {
    if (event.isSprinting()) {
        event.getPlayer().sendMessage("¡Empezaste a correr!");
    } else {
        event.getPlayer().sendMessage("Dejaste de correr.");
    }
}
```

### `ObserverPlayerIdleEvent`
> Se dispara cuando un jugador se queda inactivo o se vuelve a mover.

`com.observer.paper.api.events.ObserverPlayerIdleEvent`

**Firma**
```java
public class ObserverPlayerIdleEvent extends Event
```

**Métodos**

#### `getPlayer`
```java
public Player getPlayer()
```
Retorna: El jugador.

#### `isIdle`
```java
public boolean isIdle()
```
Retorna: `true` si el jugador entró en inactividad, `false` si volvió a moverse.

**Ejemplo**
```java
@EventHandler
public void onIdle(ObserverPlayerIdleEvent event) {
    if (event.isIdle()) {
        event.getPlayer().sendMessage("¡Estás inactivo!");
    } else {
        event.getPlayer().sendMessage("¡Te moviste!");
    }
}
```
