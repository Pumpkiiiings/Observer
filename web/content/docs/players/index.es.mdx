# Observer para Jugadores

¡Bienvenidos! Para poder jugar en un servidor que utilice Observer, necesitas instalar el mod cliente de Observer en tu juego. Este mod permite que el servidor te envíe interfaces gráficas (UI) personalizadas, sonidos ambientales y cambios de entorno.

## Requisitos Previos

- **Minecraft**: Debes usar la versión exacta de Minecraft que pide el servidor.
- **Fabric Loader**: Observer es un mod exclusivo para Fabric.

## Guía de Instalación

1. Instala el [Fabric Loader](https://fabricmc.net/use/installer/) para tu versión de Minecraft.
2. Abre la carpeta `.minecraft/mods` de tu juego.
3. Descarga el archivo `observer-fabric-<version>.jar` proporcionado por los dueños de tu servidor.
4. Coloca el archivo `.jar` dentro de la carpeta `mods`.
5. Inicia el juego seleccionando el perfil de Fabric.

## Solución de Problemas

- **El servidor me expulsa diciendo "Observer requerido"**: Asegúrate de que el mod está realmente en tu carpeta `mods` y de haber iniciado con el perfil de Fabric.
- **Texturas Faltantes (Cuadros morados y negros)**: Asegúrate de haber aceptado el "Resource Pack" del servidor al entrar. Las interfaces de Observer dependen del pack de texturas del servidor para mostrar imágenes.
- **Teclas que no funcionan**: Revisa el menú de Controles de Minecraft para asegurarte de que ninguna tecla que usa el servidor esté en conflicto con otros mods.
